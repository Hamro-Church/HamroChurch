Adding Hymns

Create new hymns directly inside Hamro Church.

Hamro Church can save new hymns into the persistent Nepali hymn JSON file so your local edits survive app restarts and future service preparation.

Best use

Local church song additions

Add songs that are not already in the imported Mandali hymn collection without editing JSON files by hand.

Open the dialog

In the Hymns drawer, click Add New Hymn. The form opens with fields for title, transliteration, lyrics, category, number, and author.

Required fields

Only the Nepali title and lyrics are required. Transliteration, hymn number, and author are optional but strongly recommended for larger libraries and bilingual operators.

Where hymns are saved

New hymns are saved to the church data file nepali_hymns.json inside the persistent app data folder. If the file does not exist yet, Hamro Church seeds it from the bundled installer resources first.
