Adding Hymns

Create new hymns and update existing ones directly in Hamro Church.

Use the same dialog to add a new hymn or edit a selected hymn from the viewer. Both actions save to the persistent local hymn file.

Best use

Local church song additions and corrections

Use this flow when you need to add a new song, fix lyrics, or correct numbering without touching JSON by hand.

Create a hymn

In the Hymns drawer, click Add New Hymn. Enter the Nepali title and lyrics, then add transliteration, category, hymn number, or author if needed.

Edit a hymn

Select a hymn in the list and click Edit in the right-side viewer. The same form opens with the current values so you can fix words, transliteration, or numbering.

Important: edits now stay saved after closing and reopening the app.

Where it is saved

New and edited hymns are written to the persistent `nepali_hymns.json` file in the local app data folder. Hamro Church only seeds that file from bundled installer resources when it is missing, so your changes are preserved.

Reset behavior

The hymns toolbar reset action restores the original bundled hymn library and removes user-created or user-edited hymn changes. Use it only when you want to return to the packaged default data.
