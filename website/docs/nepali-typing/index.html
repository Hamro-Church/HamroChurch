Nepali Typing

Type Nepali live with Roman letters inside Hamro Church.

Use the in-app typing toggle or press Ctrl+Alt+N to switch between English and live Roman-to-Nepali transliteration while editing slides, notes, or search boxes.

Best use

Slides, Bible search, and hymns search

Keep the toggle enabled when operators want Roman typing that converts immediately into Nepali Unicode.

Toggle typing

Click the floating EN / ने button in the app or press Ctrl+Alt+N. The question-mark button opens the same tips popup shown below.

Vowels

  • a -> अ
  • A or aa -> आ
  • i -> इ
  • I or ii -> ई
  • u -> उ
  • U or uu -> ऊ
  • e -> ए
  • ai -> ऐ
  • o -> ओ
  • au -> औ
  • Ri -> ऋ, and after a consonant it becomes the matra ृ

Consonants

  • k -> क्, ka -> क
  • kh or K -> ख्
  • g -> ग्, gh or G -> घ्
  • c / ch -> च्, chh -> छ्
  • j -> ज्, jh -> झ्
  • t / th / d / dh / n -> त-series
  • T / Th / D / Dh / N -> ट-series
  • p / ph / b / bh / m -> प-series

Special forms and conjuncts

  • NG -> ङ
  • NY -> ञ
  • jn -> ज्ञ
  • kSh or x -> क्ष
  • tr -> त्र
  • rr -> र्
  • rry -> र्य
  • gRi -> गृ
Predictive behavior: typing k shows क्, then typing a changes it to क automatically.

Punctuation and digits

  • . -> ।
  • | -> ।
  • .. -> .
  • 0-9 -> ०-९
