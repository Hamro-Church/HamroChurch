Bible Navigation

Move through books, chapters, and verses in Nepali or English.

Hamro Church supports Nepali display names and localized verse numbers while still letting operators work with English book names when needed.

Books

In Nepali mode, Hamro Church prefers Nepali book names for the local Nepali Bible. In English mode, it falls back to the English defaults so operators can search references naturally.

Chapters and verses

Select a book first, then choose one or more chapters and verses. Nepali mode localizes verse numbers and reference ranges using Nepali digits.

Insert to slide

Once the passage is selected, use the scripture insert flow to send the formatted text to the current presentation output or create a scripture show.
