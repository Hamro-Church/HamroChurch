Search and insert

Open the Bible panel, choose the collection you need, then search by book, chapter, and verse. Hamro Church is designed to make scripture selection fast enough for live use while still supporting careful preparation ahead of time.

  • Check the selected Bible version before inserting verses.
  • Use preview and selection tools to confirm the exact reference range.
  • Verify Nepali script rendering on the actual output screen.

Multi-version references

Hamro Church can show more than one scripture version for comparison or bilingual use. This is especially helpful for churches that project Nepali text but still want English available for operators or supporting material.

When using multi-version references, confirm the book names and verse numbering display as expected in the current language mode.

Scripture templates

Templates control how scripture appears once it is inserted into a slide or output. Create a scripture-specific template for sermon readings so long passages remain readable.

Template tip: If you are converting older scripture layouts, test one reference in the new template before converting the entire collection of saved scripture slides.

Custom Bible collections

You can import custom Nepali or English Bible files when your church uses a local or curated source. Keep naming consistent, verify every book is available, and test references from multiple sections of the Bible.

  • Store custom Bible files in a predictable church content folder.
  • Keep a known-good backup before replacing an existing collection.
  • Use the Bible Resources page as your hosted landing page for future curated downloads.
