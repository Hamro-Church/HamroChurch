Interface

Learn the workspace your operators will use every week.

Hamro Church keeps projects, shows, overlays, scripture, media, and outputs in one interface. This guide explains the main building blocks so volunteers can work confidently.

Projects

Projects group the content needed for a full service or event. Use them when you want a clear service flow that includes songs, scriptures, media, and announcements.

A simple church workflow is one project per service or event, with shows inside the project matching each segment of the order of service.

Shows

Shows are the main presentation units inside Hamro Church. A show might be a hymn, sermon scripture reading, announcement sequence, video block, or welcome slide set.

  • Keep show names clear and searchable.
  • Use categories so volunteers can browse faster.
  • Reuse existing shows instead of rewriting recurring content.

Overlays

Overlays are best for reusable lower-thirds, notices, countdown labels, or small visual elements that sit above other output content.

Use overlays when the main slide should stay unchanged but you still need an extra message or live label on top.

Templates

Templates control how content looks. Set fonts, spacing, background choices, and scripture or lyric behavior once, then reuse those settings across many shows.

For bilingual churches, create separate templates for Nepali-heavy text, English-heavy text, and mixed lower-third layouts.

Importing

Hamro Church supports practical church imports including media, scripture collections, PDFs, and presentation-style content. Test imported material once before trusting it during service.

  • Import one small sample first.
  • Check fonts and media paths after import.
  • Keep imported content in a known category or folder until it is reviewed.

Line-by-line reveal

Line-by-line reveal is useful when a worship leader wants text to appear in smaller pieces rather than an entire slide at once. It can reduce visual overload and help the congregation follow the current line.

Click reveal

Click reveal lets the operator advance parts of the same slide without leaving the current context. Use it carefully and rehearse the exact reveal timing before a live event.

Virtual breaks

Virtual breaks are layout helpers that affect how text wraps or splits without forcing the visible wording to change. They are especially useful when a hymn or scripture line needs a better presentation rhythm.

Canva and media tools

If your team uses external design tools such as Canva, treat them as content sources rather than your live control surface. Import and test the final media inside Hamro Church so output behavior stays predictable.
