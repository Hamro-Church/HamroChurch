Hymns

Browse, organize, and insert hymns with less manual cleanup.

Hamro Church is designed for churches that need reliable Nepali hymn workflows while still supporting English or bilingual presentation when required.

Use case

Prepare worship songs before rehearsal

Keep hymn content clean, categorized, and easy to search so the worship team does not fight the library during service prep.

Browse the library

The new Hymns panel uses fixed category tabs, a numbered list, and a transliteration column so operators can scan Nepali titles quickly without losing English searchability.

  • Filter by All Hymns, Bhajan, Chorus, Bal Sangati, or New Songs.
  • Search by Nepali title, transliteration, hymn number, or lyrics text.
  • Clicking a hymn now inserts it directly into the current project flow while also updating the right-side preview panel.

Insert into a show

When a hymn is selected, Hamro Church builds verse and chorus slides automatically so the worship team does not need to re-split the text manually.

Hymns layout

The hymns drawer mirrors the production-friendly layout requested by churches: category tabs on the left, a searchable hymn table in the center, and metadata plus lyrics on the right.

Operator tip: Use the transliteration column when volunteers know the hymn phonetically but not the exact Devanagari spelling.

Add new hymns

Use the Add New Hymn button in the hymns toolbar to create a new hymn with title, transliteration, lyrics, category, and hymn number. New hymns are saved into the persistent Nepali hymn JSON file in the church data folder and remain available after restart.
