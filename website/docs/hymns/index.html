Hymns

Browse, edit, and keep hymns saved for future services.

The hymns drawer now supports number search, sort buttons, transliteration, editing existing hymns, and a reset option that restores the original bundled library.

Best for

Weekly worship preparation

Use this guide when you need a fast song list, accurate Nepali titles, and confidence that changes will still be there after restart.

Browse the library

The current hymns table is designed for live church use: a compact list, a transliteration column, and sort buttons for both hymn number and hymn name.

  • Search by Nepali title, transliteration, lyrics text, or hymn number.
  • Use the No. and Name headers to sort ascending or descending.
  • The list is denser than before, so more hymns stay visible during rehearsal.

Edit and save hymns

Select a hymn, then use the Edit button in the right-side viewer. The form opens with the hymn pre-filled so you can correct lyrics, update transliteration, or fix the hymn number.

Important: hymn edits are now written to the persistent local hymn file and are no longer overwritten on the next app launch.

Add new hymns

Use Add New Hymn to create a local hymn record for songs that are not in the original library. New hymns use the same persistent storage as edited hymns, so they remain available in future sessions.

  • Store the Nepali title and lyrics at minimum.
  • Add transliteration and hymn number so volunteers can search faster.
  • Use the transliteration field for phonetic lookup when the exact spelling is not known.

Reset to original

If the local hymn library becomes messy, use the hymns toolbar Reset to original action. This discards user edits and restores the bundled Nepali hymn library and Nepali Bible that ship with the installer.
