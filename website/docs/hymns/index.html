Browse the library

Use categories, consistent naming, and hymn numbering where possible. Operators should be able to find a song quickly by title, number, or a familiar keyword without guessing which variant is correct.

  • Keep repeated songs in one canonical version when possible.
  • Use categories for Christmas, youth, communion, and special services.
  • Check that Nepali text uses the expected spelling and line breaks.

Insert into a show

When a hymn is selected, insert it into the current show and review the slide flow immediately. Confirm verse ordering, repeated choruses, and whether your chosen template handles the line lengths well.

Bilingual hymn workflow

For bilingual services, keep one slide plan that respects the congregation’s reading rhythm. Some churches prefer Nepali first with English supporting text, while others prefer separate bilingual sections. Decide on one pattern and use it consistently.

Bilingual tip: Create a dedicated lyric template for mixed-language slides so English does not overpower Devanagari text, and vice versa.

Custom hymn imports

If your church has a local hymn archive, clean the text before importing it. Remove duplicate titles, verify verse numbering, and keep backups of the source file so corrections can be applied later without retyping everything.
