Typing Tips

Roman-to-Nepali typing guide for Hamro Church operators.

Use this page to train volunteers on the in-app Nepali typing system used in slide editing, notes, and search boxes.

Shortcut

Press Ctrl+Alt+N

That shortcut toggles the live Nepali typing mode inside the desktop app. Operators can also use the floating EN / ने button.

Toggle typing

Use the in-app EN / ने toggle or press Ctrl+Alt+N to switch between normal English typing and live Roman-to-Nepali transliteration.

Best use: slide text, hymn search, notes, and scripture search fields.

Vowels

  • a -> अ
  • A or aa -> आ
  • i -> इ
  • I, ii, or ee -> ई
  • u -> उ
  • U, uu, or oo -> ऊ
  • e -> ए
  • ai -> ऐ
  • o -> ओ
  • au -> औ
  • Ri -> ऋ, and after a consonant it becomes ृ

Consonants

  • k -> क्, ka -> क
  • kh or K -> ख्
  • g -> ग्, gh or G -> घ्
  • c or ch -> च्, chh or C -> छ्
  • j -> ज्, jh or z -> झ्
  • T, Th, D, Dh, N -> ट-series
  • t, th, d, dh, n -> त-series
  • p, ph, b, bh, m -> प-series
  • y, r, l, v or w, s, sh, Sh, h -> common consonants

Special keys and conjuncts

  • NG -> ङ
  • NY -> ञ
  • jn -> ज्ञ
  • kSh or x -> क्ष
  • tr -> त्र
  • rr -> र्
  • rry -> र्य

Punctuation and digits

  • . or | -> ।
  • .. -> .
  • 0 to 9 -> ० to ९
  • M -> ं
  • ~ -> ँ
  • : -> ः

Examples

  • ka -> क
  • gRi -> गृ
  • shr -> श्र्
  • trishnaa -> तृष्णा-style consonant and matra combinations
  • sauti hos style Roman search becomes Nepali when typing mode is on
