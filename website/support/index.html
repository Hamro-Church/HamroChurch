Support

Help Hamro Church become reliable for more churches.

You do not need a formal sponsorship system to support the project. Good reports, real ministry feedback, documentation fixes, and translation help are high-value contributions.

Best next step

If you are a ministry user, report what blocks your service day. If you are technical, contribute fixes or improve the documentation.
